# Run and deploy your AI-powered portfolio

This repository contains everything you need to run the interactive portfolio (including the built-in AI assistant) locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create a `.env.local` file and set your AI provider configuration (for example:

   `AI_API_KEY=your_api_key_here`

   `AI_API_URL=https://your-ai-endpoint.example.com`

   )
3. Run the app:
   `npm run dev`
